# Add Your Hare Krishna Mantra Audio Here

## Instructions:

1. Download a Hare Krishna Maha Mantra MP3 file
2. Rename it to: `hare-krishna-mantra.mp3`
3. Place it in this folder (public/audio/)
4. Delete this README file

## Recommended Sources:

- YouTube: Search "Hare Krishna Maha Mantra" + use YouTube to MP3 converter
- ISKCON: https://www.iskcon.org/audio/
- Archive.org: Search for devotional music

## File Requirements:

- Format: MP3
- Name: `hare-krishna-mantra.mp3` (exact name)
- Quality: Any (recommend 128kbps or higher)
- Duration: 3-10 minutes (it will loop automatically)

## Testing:

After adding the file:
1. Refresh your app (Ctrl+Shift+R)
2. Login if not already logged in
3. Click anywhere on the page
4. Music should start playing
5. Use the floating control panel in bottom-right corner

The component will use a fallback placeholder sound if this file is not found.
